## Metric Definition

- Pearson: 선형 상관
- Spearman: 단조 상관 (순위 기반)
- Linear R2: 단일 feature 선형회귀 설명력

## 주석: 해석 기준 (보고서용)

- Pearson/Spearman 절대값 기준:
	- 0.0 ~ 0.1: 무시 가능한 수준의 상관
	- 0.1 ~ 0.3: 약한 상관
	- 0.3 이상: 실질적 신호 가능성 검토
- Linear R2 기준:
	- 0.05 미만: 실무적 설명력 부족
	- 0.10 이상: 보조 feature 후보
	- 0.20 이상: 주요 feature 후보

본 실험의 모든 feature는 위 기준에서 "주요 feature" 수준에 도달하지 못했습니다.

## 주석: 기존 모델 한계 보고 포인트

- 관찰된 한계:
	- 관절값 + 얼굴 면적 기반 feature는 beta 예측에 필요한 체형 신호를 충분히 제공하지 못함
	- 기존 관절 중심 4개 모델(Transformer/GCN/MLP/ResMLP)에서 MSE 0.6+ 결과와 일관된 방향

- 원인 가설:
	- 관절 좌표는 포즈/카메라 영향이 크고, 체형(둘레/부피) 정보를 직접적으로 담기 어려움
	- 얼굴 bbox 기반 feature는 거리/원근/검출 오차에 민감

- 보고 시 결론 문장 예시:
	- "현 파이프라인의 입력 신호(관절 및 얼굴 면적 파생 feature)는 체형 파라미터(beta) 회귀에 필요한 설명력이 부족하며, 입력 표현을 이미지/실루엣 중심으로 전환할 필요가 있다."

- 권장 후속 조치:
	- 관절-only 경로는 보조 feature로 축소
	- 사전학습 인체 모델(HMR/PARE/CLIFF 계열) 기반 baseline 구축
	- 전신 이미지 + 실루엣 + 관절 보조 입력의 하이브리드 구조 검증
